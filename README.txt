HelisaTest (enrutamiento simple)
Aplicacion de citas (php,mysql,jquery)

----------------
requiere
  xampp o wapp (servidor php y bd mysql)
  ||
  servidor php
  mysql


Instalacion de la aplicacion
El proyecto debe ser incluido en wampp/www o xampp/htdocs

Importe el archivo BD.sql en el gestor de bases de datos, este
debe tener el puerto 3306, con el nombre 
    citas

Dirijase a la siguiente url en el navegador(si usa wampp o xampp)
para hacer test a la app
    http://localhost/helisa/
